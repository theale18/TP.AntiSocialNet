## INSTALACIONES PREVIAS

* Node.js
Descarga: https://nodejs.org/es/download

* Visual Studio Code
Descarga: https://code.visualstudio.com/download

## DESCARGAR REPOSITORIO DE GITHUB

* Clonar Repo
- Ejecutar Git Bash
- Pegar: git clone https://github.com/theale18/TP.AntiSocialNet.git

## EJECUTAR BACKEND

* Abrir Terminal (GitBash) dentro de la Carpeta: backend
* Pegar y Ejecutar: 
- npm install
- node seed.js
- npm start

Respuesta -> Servidor corriendo en http://localhost:3001

## EJECUTAR FRONTEND

* Abrir Terminal (GitBash) dentro de la Carpeta: frontend
* Pegar y Ejecutar: 
- npm install
- npm run dev

Respuesta --> Local: http://localhost:5173
