## INSTALACIONES PREVIAS

* Node.js
Descarga: https://nodejs.org/es/download

* Visual Studio Code
Descarga: https://code.visualstudio.com/download

## DESCARGAR REPOSITORIO DE GITHUB

* Clonar Repo
- Ejecutar Git Bash
- Pegar: git clone 

## EJECUTAR BACKEND

* Abrir Terminal (GitBash) dentro de la Carpeta: backend
* Pegar y Ejecutar: 
- npm install
- node seed.js
- npm start

Respuesta: http://localhost:3001

Cerrar Puerto:
netstat -ano | findstr :3001
taskkill //PID "n° del PID"//F

## EJECUTAR FRONTEND

* Abrir Terminal (GitBash) dentro de la Carpeta: frontend
* Pegar y Ejecutar: 
- npm install
- npm run dev

Respuesta --> Local: http://localhost:5173

Cerrar Puerto:
netstat -ano | grep "5173"
taskkill //PID "n° del PID" //F
